# [프로그램 메뉴얼] AI 기반 지역별 맛집 추천 시스템 (v3.0)

**작성자:** 202644074 선현영  
**활용 AI 에이전트:** Google Gemini 기반 Antigravity AI  

---

## 1. 프로그램 개요
본 프로그램은 사용자가 입력한 지역과 메뉴 정보를 바탕으로, AI가 최적의 검색 키워드를 생성하고 실제 지리 정보 데이터를 결합하여 사용자에게 최적의 맛집 정보를 시각적으로 제공하는 **AI 융합형 유틸리티 도구**입니다.

## 2. 핵심 기술 스택 및 AI 활용 포인트
- **AI Agent (Google Gemini 1.5 Flash)**: 사용자의 추상적인 요구(예: "아무거나", "데이트 코스")를 분석하여 카카오맵 API 검색에 최적화된 구체적인 키워드로 변환합니다.
- **Data Integration (Kakao Local/Image API)**: AI가 추출한 키워드를 활용해 실제 매장의 위치, 전화번호, 상세 링크 및 대표 이미지를 실시간으로 연동합니다.
- **Cross-Platform UI (Tkinter)**: Windows와 Mac 환경 모두에서 작동할 수 있도록 설계된 GUI 프레임워크입니다.

## 3. 주요 기능 설명
1. **AI 검색 키워드 자동 생성**: 지역명과 원하는 메뉴를 입력하면 AI가 가장 높은 검색 정확도를 보장하는 키워드를 제안합니다.
2. **실시간 맛집 데이터 렌더링**: 식당의 이름, 도로명 주소, 전화번호를 한눈에 확인할 수 있습니다.
3. **상세 정보 연동**: '주소 복사' 기능과 '카카오맵 상세보기' 링크를 통해 실제 방문까지의 사용자 경험을 최적화했습니다.
4. **랜덤 추천 기능**: 결정장애가 있는 사용자를 위해 필터링된 결과 내에서 AI가 새로운 식당을 무작위로 추천해 줍니다.

## 4. 사용 방법
1. **API 키 설정**: 앱 실행 후 상단의 `⚙️ API 설정` 버튼을 눌러 본인의 Gemini 및 Kakao REST API 키를 저장합니다. (최초 1회)
2. **검색**: `📍 지역`과 `🍜 메뉴`를 입력한 후 `검색하기` 버튼을 누릅니다.
3. **결과 확인**: 하단 리스트에 출력되는 식당 정보를 확인하고, 지도를 보고 싶다면 `상세보기`를 클릭합니다.
4. **추가 추천**: 결과가 마음에 들지 않으면 `다른곳 추천받기` 버튼을 눌러 새로운 목록을 확인합니다.
